Keyboard shortcuts

[Ctrl|Cmd]-Space|Alt-Enter Trigger Autocompletion
[Ctrl|Cmd]-D Delete current/selected line(s)
[Ctrl|Cmd]-/ Comment or uncomment current/selected line(s)
[Ctrl|Cmd]-Alt-Down Copy line down
[Ctrl|Cmd]-Alt-Up Copy line up
[Ctrl|Cmd]-Shift-F Auto-format/indent selected lines
[Ctrl|Cmd]-] Indent current/selected line(s) more
[Ctrl|Cmd]-[ Indent current/selected line(s) less
[Ctrl|Cmd]-Enter Execute query or update
Shift-[Ctrl|Cmd]-Enter Execute explain plan for query
[Ctrl|Cmd]-Alt-T Create a new unnamed tab
[Ctrl|Cmd]-S Create a saved query from the current state
[Ctrl|Cmd]-Alt-Right Switch to next tab
[Ctrl|Cmd]-Alt-Left Switch to previous tab
Shift-Left-Mouse Click at close tab icon closes all the tabs except the clicked one
