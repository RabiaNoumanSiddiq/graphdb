Mouse actions

Single click View node details and properties
Double click Load node connections
Ctrl/Cmd-click Removes a node and its links
Ctrl/Cmd-Shift-click Restart the view with that node as a central one
Click and drag a node Move a node by dragging it (will also pin down the node)
Right click a node Pin down or unpin the node
Click and drag outside a node Move the whole graph

Touch actions

Tap View node details and properties
Tap and hold Removes a node and its links
Tap twice Load node connections

Keyboard actions

Left arrow Rotate the graph to the left
Right arrow Rotate the graph to the right
